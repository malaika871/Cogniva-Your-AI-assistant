WHAT YOU NEED TO RUN:

1. Ollama Server (REQUIRED)
   In terminal 1:
   $ ollama serve
   
   (If Mistral not installed yet:)
   $ ollama pull mistral

2. Streamlit App (MAIN)
   In terminal 2:
   $ streamlit run app.py --server.port=8501
   
   Or:
   $ python run.py

3. Access Browser
   Open: http://localhost:8501

TROUBLESHOOTING:

If Ollama is not running:
  Error: "Ollama is not running"
  Solution: Run 'ollama serve' in another terminal

If model not found:
  Error: "Model 'mistral:latest' not found"
  Solution: Run 'ollama pull mistral' in terminal with ollama

Port already in use:
  Error: "Address already in use"
  Solution: Use different port: streamlit run app.py --server.port=8502

Missing dependencies:
  Solution: Already verified - all present
  (streamlit, ollama, PyPDF2, python-dotenv)
